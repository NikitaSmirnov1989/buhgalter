# Сайт-визитка бухгалтерских услуг
  Данный сайт носит информационный характер - на нем представлены расценки бухгалтерских услуг
  
## Как запустить?
  1. Необходимо скачать файлы к себе на локальный компьютер при помощи команды:

  - ```git clone https://github.com/NikitaSmirnov1989/buhgalter```

  2. После скачивания проекта необходимо зайду в появившуюся папку и установить все зависимости при помощи команды:

  - ```cd buhgalter```
  - ```npm install```

  3. После установки всех зависимостей проект можно запустить локально при помощи команды:

  - ```npm start```

  4. Для создания билд версия для размещения на хостинге необходимо вызвать команду:
  
  - ```npm run build```

  ## Где размещен сайт?
Данный сайт размещен по [данному адресу](https://buhgalter-slavorossova.ru/)
